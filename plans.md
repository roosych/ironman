# Модуль: Аутентификация пользователей

## Этап 1: Полная аутентификация с Sanctum
- Установить и настроить Laravel Sanctum
- Реализовать регистрацию пользователей
- Реализовать логин (API токен)
- Реализовать восстановление пароля
- Реализовать верификацию по почте
- Создать необходимые маршруты (API v1)
- Создать контроллер AuthController в `app/Http/Controllers/Api/V1/`
- Создать FormRequest для регистрации и логина
- Создать уведомления для email верификации и восстановления пароля
- Написать Feature тесты для всех функций

### Структура версионирования API

```
app/Http/Controllers/Api/
├── V1/                          # API версия 1
│   ├── AuthController.php
│   └── [другие контроллеры v1]
├── V2/                          # API версия 2 (будущее)
│   └── ...
```

### Маршруты с версионированием

```
routes/
├── api.php                      # Основной файл, подключает версии
├── api/
│   └── v1.php                   # Маршруты API v1
```

### URL структура
- `/api/v1/auth/register`
- `/api/v1/auth/login`
- `/api/v1/auth/logout`
- и т.д.
